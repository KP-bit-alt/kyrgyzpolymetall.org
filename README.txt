HOW TO UPDATE THIS WEBSITE

1. Replace the files in your GitHub repo with the files from this folder.
2. Keep these files together in the root of the repo:
   - index.html
   - styles.css
   - script.js
   - news-data.js
   - projects-data.js
   - hero-bg.png
3. Commit the changes in GitHub.
4. Vercel will redeploy the site automatically.

HOW TO ADD NEWS
- Open news-data.js
- Add a new object inside the newsItems array using the example format shown in the file.
- Save and push to GitHub.

HOW TO ADD PROJECTS
- Open projects-data.js
- Add a new object inside the projectItems array using the example format shown in the file.
- Save and push to GitHub.

IMPORTANT
- Right now, all projects appear in the Projects section.
- Right now, all news items appear in the News section.
